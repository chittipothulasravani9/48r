1. What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It structures web content and is the foundation of all websites on the internet.

2. What are semantic HTML elements?

Semantic HTML elements clearly describe their meaning in a human- and machine-readable way. Examples include <article>, <section>, <header>, <footer>, and <nav>.

3. What is the difference between <div> and <span>?

<div> is a block-level element used for grouping larger chunks of content, while <span> is an inline element used for styling small parts of text or elements within a line.

4. What is the purpose of the <DOCTYPE> declaration?

The <!DOCTYPE> declaration defines the document type and version of HTML being used. It helps the browser render the page correctly.

5. What are void elements in HTML?

Void elements are self-closing HTML elements that do not require a closing tag. Examples include <img>, <br>, <hr>, and <input>.

6. What is the difference between id and class in HTML?

The id attribute is used to uniquely identify a single element on a page, while the class attribute is used to group multiple elements for styling or scripting.

7. How do you create a hyperlink in HTML?

You create a hyperlink using the <a> tag. Example: <a href="https://example.com">Visit Example</a>.

8. What is the purpose of the <meta> tag?

The <meta> tag provides metadata about the HTML document, such as character encoding, author, description, and viewport settings for responsive design.

9. What is the difference between HTML and XHTML?

HTML is more forgiving with syntax errors, while XHTML is stricter and based on XML, requiring well-formed code with properly closed tags and case sensitivity.

10. What is the role of the <alt> attribute in images?

The alt attribute provides alternative text for an image if it cannot be displayed. It also improves accessibility for screen readers and contributes to SEO.
